安装过程

下载镜像

制作U盘启动

windows系统下可以使用transMac工具制作

配置clover

这应该是安装黑苹果最困难年的地方吧。我在网上下载了几个硬件配置与我相近的网友的配置文件，根据自己的硬件情况进行微调，这是我目前使用的配置文件。

U盘安装

安装本身比较简单，按提示操作即可。但实际操作过程中会遇到报错：

不能验证这个“安装 OS X EI Capitan”应用程序副本，它在下载过程中可能一遭破坏和篡改。

这是苹果公司更换证书导致的，只需要将当前系统时间更改为2015.12.20-2016.2.14之间即可。

安装N卡驱动

u盘安装后进入苹果系统，这时除了显卡外其他设备应该都正常驱动，在官方下载最新N卡驱动，直接安装会提示与苹果系统版本不匹配，无法安装。

github上有相应解决方法。使用本地安装方式安装即可。

至此，黑苹果基本能正常使用了。后面就是长期使用过程中的微调。我用了半天，各种不习惯，基本不再登录。
